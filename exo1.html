La formation nous apprend les technologies suivantes:

  3. JavaScript

    • Langage de programmation (orienté objet) communément utilisé pour dynamiser les sites internet.

  4. PHP

    • Langage de programmation (scripting) qui est utilisé pour dynamiser le contenu des pages HTML

  5. Base de données (SQL)

    • C'est grâce à ces dernières que le contenu dynamiquement généré et stocké. Le langage généralement utilisé pour dialoguer avec la base de données.
